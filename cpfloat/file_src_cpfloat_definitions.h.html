Definition (src/cpfloat_definitions.h)

Detailed Description

This file includes all the external header files used by CPFloat, defines the enumerated types

  • Enum cpfloat_subnormal_t,

  • Enum cpfloat_explim_t,

  • Enum cpfloat_rounding_t,

  • Enum cpfloat_softerr_t,

and the structured data type Struct optstruct. It is not necessary to include this file in order to use CPFloat, as it is already included by File cpfloat_binary32.h and by File cpfloat_binary64.h.
